.Element

ZRender 的 Element 元素类。

查看源码

官方 APIs 文档

状态​

一系列与元素实例状态管理相关的 APIs。

示例：

// * 为元素实例添加一个鼠标的 hover 效果
// 为元素注册一个 hover 状态对象
el.states.hover = {
  style: {
    fill: 'red',
  },
};

// 设置元素的状态转换动画
el.stateTransition = {
  duration: 400,
  done: () => {
    console.log('switch state done!');
  },
};

// 添加鼠标事件
el.onmouseover = function () {
  el.useState('hover', true);
};
el.onmouseover = function () {
  el.removeState('hover');
};
