该网站使用 [Docusaurus 2](https://docusaurus.io/) 构建, 一个现代的静态站点生成器。

### 安装

```bash
$ npm install
```

### 本地开发

```bash
$ npm run start
```

此命令启动本地开发服务器并打开浏览器窗口。大多数更改都会实时反映，而无需重新启动服务器。

### 构建

```bash
$ npm run build
```

此命令将静态内容生成到 `build` 目录中，并且可以使用任何静态内容托管服务提供服务。
